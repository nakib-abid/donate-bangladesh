What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML document as a tree of objects, allowing developers to interact with and manipulate the content, structure, and style of web pages. It enables dynamic changes to the document through JavaScript, allowing for features like updates, animations, and event handling.

How do you select an element from the DOM?

To select an element from the DOM, I can use methods like document.getElementById('elementId') for single elements by ID, or document.querySelector('selector') for selecting using CSS selectors. I can also use document.getElementsByClassName('className') to get a live HTMLCollection of elements with a specific class name. For selecting multiple elements, document.querySelectorAll('selector') returns a static NodeList of matching elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of individual listeners on each child element. This allows events triggered by child elements to be handled by the parent, improving performance and reducing memory usage.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes in the DOM, use getAttribute and setAttribute methods to read and update attributes, respectively. For styles, directly modify the style property of the element to change CSS styles. We can access elements using methods like getElementById or querySelector.
